# minimin-package-serverstats

A server statistics package based around system information.

Current information this provides:
- CPU Information
  - Model
  - Vendor
  - Speed (Mhz)
- RAM Information
  - Type
  - Total
  - Free
- HDD Information
  - Name
  - Vendor
  - Size
  - Device Path

Designed for the [Minimin](https://github.com/pxgamer/minimin) server management tool.
